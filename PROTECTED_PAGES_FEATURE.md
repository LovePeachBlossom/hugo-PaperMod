# 🔒 密码保护文章分类功能

## ✅ 功能已实现

现在您可以在右上角菜单中看到 **"🔒 密码保护"** 菜单项，点击后会显示所有需要密码才能查看的文章。

## 📋 实现内容

### 1. 新增页面
- **页面路径**：`/protected/`
- **文件位置**：`content/protected.md`
- **布局模板**：`layouts/_default/protected.html`

### 2. 菜单项
- **菜单名称**：🔒 密码保护
- **菜单位置**：右上角导航栏
- **菜单顺序**：在"搜索"和"归档"之后（weight: 30）

### 3. 功能特点

✅ **自动筛选**：自动显示所有有 `passwordHash` 参数的文章  
✅ **密码提示**：在列表页显示密码提示信息（如果有）  
✅ **锁图标**：每篇文章显示 🔒 图标标识  
✅ **分页支持**：如果文章较多，支持分页显示  
✅ **空状态提示**：如果没有密码保护文章，显示友好提示  

## 🎯 使用方法

### 访问密码保护文章列表

1. **通过菜单**：点击右上角菜单中的 "🔒 密码保护"
2. **直接访问**：访问 `/protected/` 路径

### 文章筛选逻辑

页面会自动筛选所有满足以下条件的文章：
- 文章有 `passwordHash` 参数
- `passwordHash` 不为空

### 显示内容

每篇文章会显示：
- 文章标题（带 🔒 图标）
- 文章摘要
- 密码提示（如果有设置）
- 文章元数据（日期、标签等）

## 📝 配置说明

### 菜单配置

在 `hugo.yaml` 中已添加：

```yaml
menu:
  main:
    - identifier: protected
      name: 🔒 密码保护
      url: /protected/
      weight: 30
```

### 文章配置

为文章添加密码保护：

```yaml
---
title: "我的私密文章"
passwordHash: "生成的哈希值"
passwordHint: "密码提示（可选）"
---
```

## 🔧 自定义

### 修改页面标题

编辑 `content/protected.md`：

```yaml
---
title: "您的自定义标题"
---
```

### 修改筛选逻辑

编辑 `layouts/_default/protected.html`，修改筛选条件：

```go
{{- $protectedPages := where site.RegularPages "Params.passwordHash" "!=" nil }}
```

### 修改菜单名称

编辑 `hugo.yaml`：

```yaml
menu:
  main:
    - identifier: protected
      name: 您的菜单名称
      url: /protected/
      weight: 30
```

## ✨ 使用示例

1. **创建密码保护文章**
   ```yaml
   ---
   title: "会员专享内容"
   passwordHash: "1234567890"
   passwordHint: "提示：我的生日"
   ---
   ```

2. **访问密码保护列表**
   - 点击菜单 "🔒 密码保护"
   - 查看所有密码保护文章

3. **查看文章**
   - 点击文章标题
   - 输入密码解锁内容
